# README.md

Template repo "starter beads + task.md".

Nội dung:
- task.md: roadmap mẫu
- worklog.md: nhật ký tùy chọn
- .beads/: thư mục Beads (đặt sẵn placeholder)

Cách dùng:
1) Cài beads CLI (bd)
2) Trong repo mới:
   bd init --prefix <prefix-ngan>
3) Dùng TASKS_GUIDE.md để chọn mức áp dụng

Lưu ý: Không commit file nhạy cảm. Chỉ commit .beads/ nếu muốn sync task.
