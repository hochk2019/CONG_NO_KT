# UX_NAVIGATION_MAP - Cong No Golden

## Main navigation
- Dashboard
- Customers
- Import Wizard
- Advances
- Receipts
- Reports
- Admin (Users/Roles, Period Locks, Audit)

## Key flows
- Login -> Dashboard
- Import Wizard: Upload -> Preview -> Commit -> History
- Receipts: Create draft -> Preview allocations -> Approve
- Advances: Create -> Approve -> Void/Reversal
- Reports: Filters -> View -> Export
- Admin: Manage users/roles, lock/unlock periods, view audit logs
